# OIBSIP
Internship Technical Assessment - Oasis Infobyte

Data Science Internship Projects
This repository contains my solutions for three machine learning tasks completed during my Data Science internship. Each project demonstrates a different aspect of the machine learning pipeline, including data cleaning, feature engineering, model training, and performance visualization.
🚀 Projects Overview
1. Iris Flower Classification
• Goal: Classify Iris flowers into three species (Setosa, Versicolor, and Virginica) based on sepal and petal measurements.
• Model Used: K-Nearest Neighbors (KNN) Classifier.
• Key Features: * Automated encoding of species categories.
• Visual evaluation using a Confusion Matrix heatmap.
• Detailed classification report (Precision, Recall, F1-Score).
2. Car Price Prediction
• Goal: Predict the selling price of used cars based on factors like mileage, fuel type, and transmission.
• Model Used: Linear Regression.
• Key Features:
• Feature engineering: Converted 'Year' into a 'Car_Year_Usage' column for better model relevance.
• Categorical encoding for fuel and transmission types.
• Performance visualization comparing Actual Price vs. Predicted Selling Price.
3. Sales Prediction (Advertising Data)
• Goal: Forecast sales based on advertising spend across TV, Radio, and Newspaper channels.
• Model Used: K-Nearest Neighbors (KNN) Regressor.
• Key Features:
• Analysis of the impact of different media on total sales.
• Model evaluation using R^2 Score and Mean Absolute Error (MAE).
• Visual representation of prediction accuracy through scatter plots.
🛠️ Technologies Used
• Language: Python
• Libraries: * Pandas & NumPy (Data Manipulation)
• Scikit-learn (Machine Learning & Evaluation)
• Matplotlib & Seaborn (Data Visualization)
